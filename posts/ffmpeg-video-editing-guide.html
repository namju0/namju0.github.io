ffmpeg로 동영상 편집하기 - 포맷 변환부터 구간 자르기까지

ffmpeg는 무료이면서도 강력한 동영상 편집 도구로 널리 사용되고 있습니다. 이번 포스팅에서는 ffmpeg를 사용하여 동영상 포맷을 변경하거나 화질을 조정하고, 화면을 크롭하거나 구간을 자르는 등 기본적인 작업을 수행해 봅니다. 여러 영상을 합치는 방법과 동영상에서 소리만 추출하는 만드는 방법도 함께 알아보겠습니다.

1. 동영상 포맷 변경

동영상 파일의 포맷을 변경하는 방법은 매우 간단합니다. ffmpeg의 기본 명령어를 사용해 입력 파일의 확장자와 출력 파일의 확장자를 지정해주기만 하면 됩니다.

ffmpeg -i input.mp4 output.avi

위 명령어는 input.mp4를 AVI 포맷으로 변환하여 output.avi로 저장합니다.

2. 포맷은 유지하면서 화질 변경

동영상 화질을 낮추거나 높이기 위해 비트레이트를 조정할 수 있습니다. 비트레이트는 높을수록 화질이 좋아지지만 그만큼 파일 크기 커진다는 것을 참고해 주세요.

ffmpeg -i input.mp4 -b:v 1000k output.mp4

이 명령어는 비트레이트를 1000kbps로 설정하여 output.mp4로 저장합니다.

3. 동영상 화면 자르기(크롭, crop)

영상의 특정 부분만 잘라내고 싶을 때는 -vf crop 필터를 사용합니다. 다음 명령어는 640x480 크기로 화면을 잘라냅니다.

ffmpeg -i input.mp4 -vf "crop=640:480:100:50" output.mp4

여기에서는 왼쪽으로부터 100px, 위로부터 50px을 시작점으로 640x480 크기를 잘라내어 비디오를 만듭니다.

4. 구간 자르기(컷 편집)

동영상의 특정 구간만 잘라내는 방법은 -ss(시작 시간)과 -to(끝 시간) 옵션을 사용합니다.

ffmpeg -i input.mp4 -ss 00:01:00 -to 00:02:00 -c copy output.mp4

이 명령어는 1분에서 2분 사이의 구간만 잘라 output.mp4로 저장합니다.

5. 여러 영상 합치기

여러 동영상을 하나로 이어 붙여 합칠 때는 파일 목록을 .txt 파일로 만들어 이를 기반으로 처리합니다.

  1. 파일 목록을 만드세요:
file 'part1.mp4'
file 'part2.mp4'
file 'part3.mp4'
  1. 이어서 다음과 같이 실행하세요:
ffmpeg -f concat -safe 0 -i file_list.txt -c copy output.mp4

이렇게 하면 파일 목록에 나열된 동영상을 순서대로 이어 붙어 합쳐진 output.mp4이 생성됩니다.

6. 동영상에서 소리만 추출해 MP3 만들기

동영상에서 오디오만 추출하려면 -vn(비디오 제외) 옵션을 사용하여 비디오를 제외하고 오디오만 남깁니다.

ffmpeg -i input.mp4 -vn -acodec mp3 output.mp3

이렇게 함으로써 동영상에서 소리만 추출하여 MP3 파일로 만들 수 있습니다.

ffmpeg는 무료이지만 매우 강력한 동영상 편집 도구로, 다양한 형식의 동영상 파일을 쉽게 변환하거나 편집할 수 있습니다. ffmpeg는 훨씬 많은 복잡한 기능을 갖고 있지만 우리가 자주 사용하게 되는 것만 일부 소개해 보았습니다. ffmpeg는 커맨드라인 기반(CLI)이라 사용이 다소 어려울 수 있지만 필요한 이용법만 기억했다 사용하면 GUI 기반 툴보다 오히려 편리할 수 있습니다.
